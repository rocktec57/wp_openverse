# Media tag

By default, the component is a transparent wrapper over `<a>` so all attributes
of the anchor tag element can be applied to it, e.g. `href`.

<Canvas>
  <Story
    name="Default"
    args={{

  >
    {Template.bind({})}
  </Story>
</Canvas>

The root node of the tag can be overridden to a different HTML tag or a Vue
component, e.g. `NuxtLink`.
